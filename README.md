# Cebu Trip

Mobile-first travel booklet for a Cebu trip.

## What This Repo Contains

- `Cebu_trip_booklet/index.html`: main booklet page
- `Cebu_trip_booklet/styles.css`: visual design and responsive layout
- `Cebu_trip_booklet/script.js`: itinerary, flight info, maps, media, and guide data
- `Cebu_trip_booklet/requirements.md`: initial planning notes

## Open Locally

Open this file in a browser:

`Cebu_trip_booklet/index.html`

## Main Features

- Mobile-first one-page travel booklet
- Flight section with outbound and return details
- Day-by-day itinerary cards
- Quick Google Maps links
- Visual cards, YouTube references, and travel guides
- Packing and budget notes
